MailChimp API Project using Node.js

Though as the end-user you cannot see the back-end functionality of this project, please feel free to view the hosted website and "sign up for my newsletter" here. This is a simple project I created to showcase that I can set up a functional API connection. I built this web page using Node and Express.js, as well as Bootstrap for styling. When the user inputs their information, it is instantly sent through to my MailChimp account, where I can actively send them a newsletter, marketing e-mails, etc. for a hypothetical business or website.
