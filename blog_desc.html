Personal Blog Website

(Work in Progress)
Project using Node.js and MongoDB

This project is currently in progress, but please check out what I have so far! I have completed all of the styling and content using Node and Express.js, and am currently working on implementing MongoDB to store data. It is designed as a personal blog, but for now I have included the "compose" page for visitors to play with in order to view the functionality of the site.

In its current state, the home page looks quite empty at first. This is because there are no posts yet! In the "Compose" page, we are given a form with a space to choose a title for our blog entry, and a space to type out the contents of the blog post.

Once we publish the post, we are re-directed to the home page, where a preview to the blog post can be seen.
After publishing multiple blog posts, the below image is what the home page would look like.

When the user clicks the "Read More" link following each blog preview, they are re-directed to a new page containing the entire blog post.

Once my database set-up is complete, the page will be pre-populated with blog posts that have already been supplied by MongoDB, and will continue to be updated as more posts are added!
